# 配信録画くん

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg) ![Platform](https://img.shields.io/badge/platform-Windows-0078D6.svg) ![License](https://img.shields.io/badge/license-MIT-green.svg)

**お気に入りの配信を、もう見逃さない。** URLをポンと入れるだけで、誰でも簡単に高画質録画ができるWindows用ツールです。推しの「今」を、ずっとあなたの手元に。

![スクリーンショット](screenshot.png)

## 特徴 (Features)

* **📺 マルチプラットフォーム対応** YouTube, Twitch, Kick, radiko, bilibili, OPENREC.tv など、主要な配信サイトに対応。URLを貼るだけでサイトを自動認識します。
* **⚡ 超軽量・高速動作** PCへの負荷を最小限に抑えた設計。ゲームプレイ中や作業中のバックグラウンド録画でも邪魔になりません。
* **📼 自動録画・再開機能** 配信が途中で切断されても、自動的に再接続して録画を継続する「自動録画再開」機能を搭載。長時間の監視録画も安心です。
* **🎨 シンプルなUI** 複雑な設定は不要。直感的に使えるポップでかわいいデザインを採用しました。
* **📂 MP4形式で保存** 汎用性の高いMP4 (H.264) で保存されるため、録画後すぐにスマホへ転送したり、動画編集ソフトで扱えます。
* **🔒 完全ローカル処理** 録画データはすべてあなたのPC内に保存されます。外部サーバーを経由しないため、プライバシー面でも安心です。

## 使い方 (Usage)

1.  **URLを入力**: 録画したい配信ページのURLをコピーし、アプリの入力欄に貼り付けます。
2.  **録画開始**: 「録画開始」ボタンを押すと、録画が始まります。
3.  **保存**: 録画を停止すると、指定したフォルダにMP4ファイルが保存されます。
